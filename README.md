#  MathStack

> Sistema educativo enfocado en la comprensión y reforzamiento de contenidos matemáticos universitarios.

MathStack es una aplicación móvil desarrollada para ayudar a estudiantes universitarios a identificar deficiencias académicas en matemáticas, generar rutas personalizadas de aprendizaje y mejorar su desempeño mediante contenido interactivo y gamificación.

---

## 🎯 Objetivo

Desarrollar una aplicación móvil de aprendizaje adaptativo que permita identificar deficiencias académicas en matemáticas, generar rutas personalizadas de estudio y fortalecer las habilidades matemáticas de estudiantes universitarios.

---

## ✨ Características

- 📊 Evaluación diagnóstica inicial.
- 🧠 Rutas de aprendizaje personalizadas.
- 📚 Lecciones de Álgebra, Aritmética y Cálculo.
- 🎮 Sistema de gamificación.
- 🔥 Rachas de estudio.
- 🏆 Insignias y recompensas.
- 💰 Monedas virtuales.
- 📈 Seguimiento de progreso.
- 🔔 Notificaciones.
- 👥 Retos entre usuarios.
